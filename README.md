# 🎨 Appwise Academy | Prácticas y Desafíos de CSS3

¡Bienvenidos al módulo de Diseño y Estilos Web!

Si en el módulo anterior construimos los cimientos y las paredes de nuestra casa con HTML, en este repositorio **vamos a pintar, decorar y acomodar los muebles**.

CSS es el lenguaje que le da vida, color y estructura visual a la web. Aquí no construiremos proyectos completos desde cero todavía; en cambio, nos enfocaremos en dominar conceptos específicos a través de un sistema de **"Teoría + Katas (Mini-retos)"**.

---

## 🛠️ ¿Cómo usar este repositorio? (Flujo de Trabajo)

Al igual que en HTML, **NO** clones este repositorio directamente. Sigue estos pasos:

1. Haz clic en el botón verde **"Use this template"** (Usar esta plantilla) arriba a la derecha.
2. Selecciona **"Create a new repository"**.
3. Nómbralo (ej: `mi-nombre-appwise-css`) y ponlo en Privado o Público.
4. Clónalo en tu computadora desde la terminal:
   ```bash
   git clone <URL-DE-TU-NUEVO-REPOSITORIO>
   ```
5. Ábrelo en Visual Studio Code.

---

## 📁 Estructura del Repositorio (El Temario)

Este módulo está diseñado para que aprendas un concepto con el código del profesor, y luego lo apliques inmediatamente en un reto aislado.

```plaintext
📦 Curso-CSS-Appwise
┣ 📂 01-Teoria-Box-Model/       (Apunte: Selectores, colores y márgenes)
┣ 📂 02-Reto-Botones/           (Práctica: Diseña 3 botones UI)
┣ 📂 03-Teoria-Flexbox/         (Apunte: Alineando elementos en 1 dimensión)
┣ 📂 04-Reto-Navbar/            (Práctica: Construye una barra de navegación)
┣ 📂 05-Teoria-Grid/            (Apunte: Maquetación avanzada en 2 dimensiones)
┣ 📂 06-Reto-Galeria/           (Práctica: Galería de imágenes estilo Pinterest)
┣ 📂 07-Teoria-Posicionamiento/ (Apunte: Relative, Absolute, Fixed y Sticky)
┣ 📂 08-Reto-Modal/             (Práctica: Centrar una ventana emergente)
┗ 📂 09-Proyecto-Final/         (Instrucciones para estilizar tus proyectos HTML previos)
```

---

## 📜 Reglas del Juego

- **Las carpetas "Teoría" son tus apuntes:** Ábrelas, lee el `README.md` y revisa el archivo `style.css` donde el profe dejó todos los comentarios explicativos. ¡Úsalas como machete!

- **Las carpetas "Reto" son tu tarea:** Ahí solo encontrarás un HTML sin gracia. Tu misión es crear la magia en el archivo CSS siguiendo las instrucciones.

- **Usa el Inspector de Elementos (F12):** En CSS, tu mejor amigo no es solo MDN, es la herramienta de desarrollador de tu navegador. Si algo no cuadra, ¡inspecciónalo!
